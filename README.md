# Inventory Manager 🚀

## Pasos para instalar:
1. **Clonar:** `git clone https://github.com/tu-usuario/inventory-manager.git`
2. **Instalar dependencias:** `composer install`
3. **Configurar entorno:** Copiar `.env.example` a `.env` y configurar tu DB.
4. **Generar clave:** `php artisan key:generate`
5. **Crear tablas y datos:** `php artisan migrate --seed`
6. **Levantar:** `php artisan serve`
